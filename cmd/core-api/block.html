HTTPS note

Direct HTTPS access to a blocked third-party domain may still trigger a browser certificate warning before this page can render. That is expected: TLS validates the original domain name before Safe Zone can show the block explanation.

Safe Zone keeps the block page generic because DNS sinkholing does not always have access to the original verdict metadata at HTTP render time.
